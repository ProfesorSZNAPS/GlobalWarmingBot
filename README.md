#🌍 Discord Bot – Zmiany Klimatyczne

Prosty bot Discord napisany w Pythonie, który edukuje użytkowników na temat zmian klimatycznych poprzez rozmowę oraz quiz.

📌 Funkcje

Bot oferuje dwie główne funkcjonalności:

💬 /rozmowa

Pozwala użytkownikowi wybrać temat związany ze zmianami klimatycznymi i otrzymać krótkie wyjaśnienie.

Dostępne tematy:

Czym są zmiany klimatyczne
Jak poważny jest problem i czy dotyczy Polski
Co powoduje zmiany klimatyczne
Jak można je powstrzymać
🧠 /quiz

Interaktywny quiz sprawdzający wiedzę użytkownika.

3 pytania jednokrotnego wyboru
15 sekund na odpowiedź na każde pytanie
Punktacja:
✅ poprawna odpowiedź: +1
❌ błędna odpowiedź: -1
ℹ️ /bothelp

Wyświetla listę dostępnych komend i ich opis.

⚙️ Wymagania
Python 3.8+
Biblioteka discord.py

Instalacja zależności:

pip install discord.py
🚀 Uruchomienie
Skopiuj kod do pliku, np. bot.py
Wklej swój token bota w miejsce:
bot.run("TWÓJ_TOKEN")
Uruchom bota:
python bot.py
